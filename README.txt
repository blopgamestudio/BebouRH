# C21 ORBAT RH Web

Ouvre `index.html` dans ton navigateur.

Contenu généré depuis `Orbat Marines.xlsx` :
- 48 personnages principaux
- 8 personnages secondaires
- 19 grades détectés
- 5 formations détectées
- feuilles Excel converties en grilles lisibles

La fausse BDD sauvegarde les changements en localStorage. Le bouton Exporter JSON permet de récupérer les modifications.

V5: Médailles/décorations passées en cases à cocher pour création et édition personnel.
